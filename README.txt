This simple php script can be used to get lyrics song easly using lyricswiki.
Simply upload that script in a web directory,and you will have an url like that:
http://example.com/lyrics.php
How to make a reques:
Use a http_get method and post that value:
artist
song
So you will have an url like that:
http://example.com/lyrics.php?artist=Metallica&song=Battery

Enjoy ;)
